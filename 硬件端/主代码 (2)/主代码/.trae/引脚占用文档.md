# 项目引脚占用文档

## 概述
本文档记录了STM32F103C8项目中所有硬件模块的引脚占用情况，方便后续开发和维护。

## 引脚占用表

### GPIOA 引脚
| 引脚 | 功能 | 模块 | 备注 |
|------|------|------|------|
| PA0 | 触摸按键A | TOUCH_KEY | 外部中断触发 |
| PA1 | 触摸按键B | TOUCH_KEY | 普通输入 |
| PA2 | 触摸按键C | TOUCH_KEY | 外部中断触发 |
| PA3 | 触摸按键D | TOUCH_KEY | 外部中断触发 |
| PA9 | USART1_TX | Serial | 串口1发送 |
| PA10 | USART1_RX | Serial | 串口1接收 |
| PA13 | 继电器2 | RELAY | 输出控制 |
| PA14 | 继电器1 | RELAY | 输出控制 |

### GPIOB 引脚
| 引脚 | 功能 | 模块 | 备注 |
|------|------|------|------|
| PB6 | SCL | OLED | I2C时钟线 |
| PB7 | SDA | OLED | I2C数据线 |
| PB12 | LED1 | LED | 输出控制 |
| PB13 | LED2 | LED | 输出控制 |

## 模块详细信息

### 1. 继电器模块 (RELAY)
- **功能**：控制外部设备供电
- **引脚**：
  - RELAY1: GPIOA_Pin_14
  - RELAY2: GPIOA_Pin_13
- **控制方式**：高电平触发
- **应用**：控制水泵等外部设备

### 2. 触摸按键模块 (TOUCH_KEY)
- **功能**：用户输入
- **引脚**：
  - TOUCH_KEY_A: GPIOA_Pin_0
  - TOUCH_KEY_B: GPIOA_Pin_1
  - TOUCH_KEY_C: GPIOA_Pin_2
  - TOUCH_KEY_D: GPIOA_Pin_3
- **触发方式**：低电平触发
- **中断配置**：A、C、D键支持外部中断

### 3. LED模块
- **功能**：状态指示
- **引脚**：
  - LED1: GPIOB_Pin_12
  - LED2: GPIOB_Pin_13
- **控制方式**：低电平点亮

### 4. OLED模块
- **功能**：显示信息
- **引脚**：
  - SCL: GPIOB_Pin_6
  - SDA: GPIOB_Pin_7
- **通信方式**：I2C

### 5. 串口模块 (Serial)
- **功能**：数据通信
- **引脚**：
  - TX: GPIOA_Pin_9
  - RX: GPIOA_Pin_10
- **通信参数**：115200bps, 8N1

## 注意事项
1. 继电器使用了PA13和PA14引脚，这两个引脚原本是JTAG/SWD调试接口，使用时需要禁用JTAG功能
2. 触摸按键使用了PA0、PA2、PA3引脚，配置了外部中断
3. OLED使用了PB6和PB7引脚，通过软件模拟I2C通信
4. 串口1使用了PA9和PA10引脚，用于调试和数据通信
