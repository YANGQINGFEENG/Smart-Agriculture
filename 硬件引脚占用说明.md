## 🔌 引脚总览

### GPIOA 引脚占用

| 引脚 | 功能 | 模式 | 外设/模块 | 说明 |
|------|------|------|-----------|------|
| PA0 | PWM输出 | 复用推挽输出 | TIM2_CH1 | PWM输出（舵机控制） |
| PA0 | 触摸按键A | 上拉输入 | 触摸按键 | ⚠️ **冲突** - 与TIM2_CH1冲突 |
| PA1 | 触摸按键B | 上拉输入 | 触摸按键 | 触摸按键B输入 |
| PA2 | WiFi TX | 复用推挽输出 | USART2 | WiFi模块串口发送 |
| PA2 | 触摸按键C | 上拉输入 | 触摸按键 | ⚠️ **冲突** - 与USART2_TX冲突 |
| PA3 | WiFi RX | 浮空输入 | USART2 | WiFi模块串口接收 |
| PA3 | 触摸按键D | 上拉输入 | 触摸按键 | ⚠️ **冲突** - 与USART2_RX冲突 |
| PA4 | WiFi RST | 推挽输出 | WiFi模块 | WiFi模块复位控制 |
| PA4 | DHT11数据 | 推挽输出 | DHT11 | ⚠️ **冲突** - 与WiFi RST冲突 |
| PA5 | 光照传感器 | 模拟输入 | ADC1_CH5 | 光照强度采集 |
| PA5 | 电机方向控制 | 推挽输出 | 电机驱动 | ⚠️ **冲突** - 与ADC冲突 |
| PA8 | RS485方向控制 | 推挽输出 | RS485 | RS485收发方向控制 |
| PA9 | 调试串口TX | 复用推挽输出 | USART1 | 串口调试输出 |
| PA10 | 调试串口RX | 上拉输入 | USART1 | 串口调试输入 |
| PA13 | 继电器2 | 推挽输出 | 继电器 | 水泵控制 |
| PA14 | 继电器1 | 推挽输出 | 继电器 | 风扇控制 |

## ⚠️ 引脚冲突分析

### 严重冲突（需要立即解决）

| 冲突引脚 | 冲突模块 | 冲突说明 | 建议解决方案 |
|----------|----------|----------|--------------|
| PA0 | TIM2_CH1 / 触摸按键A | PWM与触摸按键冲突 | 将触摸按键A移至PA15 |
| PA2 | USART2_TX / 触摸按键C | WiFi串口与触摸按键冲突 | 将触摸按键C移至PA6 |
| PA3 | USART2_RX / 触摸按键D | WiFi串口与触摸按键冲突 | 将触摸按键D移至PA7 |
| PA4 | WiFi RST / DHT11 | WiFi复位与温湿度传感器冲突 | 将DHT11移至PA11 |
| PA5 | ADC1_CH5 / 电机方向 | 光照传感器与电机控制冲突 | 将电机方向移至PA12 |
| PC13 | 按键KEY0 / LED | 按键与LED冲突 | 将LED移至PC14 |

## 📌 注意事项

1. **引脚冲突**：当前存在多处引脚冲突，需要重新规划引脚分配
2. **电源供应**：确保WiFi模块和继电器模块有足够的电流供应
3. **信号完整性**：高速信号（如USART）走线尽量短
4. **EMC设计**：继电器控制需要增加续流二极管
5. **调试接口**：PA9/PA10用于调试，不要占用
